# WoW Combat Live Analytics

A real-time data pipeline that streams World of Warcraft combat logs from a PC to a Raspberry Pi for live visualization.

## Features
- **Data Streaming:** Python-based file tailing and HTTP streaming.
- **Backend:** Flask API on Raspberry Pi with SQLite persistence.
- **Frontend:** Responsive Web-Dashboard using Chart.js.

## Tech Stack
- Python (Flask, Requests, SQLite3)
- JavaScript (Chart.js, Fetch API)
- Hardware: Raspberry Pi 3
